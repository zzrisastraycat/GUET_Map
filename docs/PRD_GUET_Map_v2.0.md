## 1. 产品概述

### 1.1 产品背景

GUET Map 是一款面向桂林电子科技大学（Guilin University of Electronic Technology）师生的校园导航应用。当前版本已完成地图显示、地点搜索、定位展示、指引导航添加等基础功能。随着用户需求的增长，亟需扩展智能化、社交化功能，提升用户体验。

### 1.2 产品定位

打造集 **智能导航**、**AI 助手**、**校园服务**、**社交互动** 于一体的智慧校园超级App。

### 1.3 现有功能清单

| 功能模块 | 描述 | 技术实现 |
|---------|------|---------|
| 地图显示 | 校园地图展示与交互 | 高德地图 SDK |
| 搜索地点 | 关键词搜索校园地点 | 高德 POI 搜索 |
| 显示定位 | 实时显示用户当前位置 | 系统定位 + 高德定位 |
| 指引导航添加 | 地点图文指引提交 | API + 本地存储 |

## 5. 技术方案概述

### 5.1 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         客户端 (Android)                         │
├─────────────────────────────────────────────────────────────────┤
│  UI层     │  Activity/Fragment │ ViewModel │ Compose UI        │
├───────────┼─────────────────────────────────────────────────────┤
│  业务层   │  Repository │ UseCase │ 业务逻辑                    │
├───────────┼─────────────────────────────────────────────────────┤
│  数据层   │  Local(Room) │ Remote(Retrofit) │ 缓存策略         │
├───────────┼─────────────────────────────────────────────────────┤
│  基础层   │  DI(Hilt) │ 网络 │ 定位 │ 地图 │ 推送              │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 关键技术选型

| 模块 | 技术选型 | 说明 |
|-----|---------|------|
| 地图 | 高德地图 Android SDK | 导航/定位/POI |
| AI对话 | 文心一言/通义千问 API | 智能问答 |
| 实时通信 | WebSocket / Firebase Realtime DB | 好友定位/聊天 |
| 推送 | Firebase Cloud Messaging | 公告/提醒 |
| 存储 | Room + DataStore | 本地持久化 |
| 图片 | Coil + OSS | 图片加载与存储 |
| 分享 | 微信/QQ 开放SDK | 社交分享 |

## 6. 非功能性需求

### 6.1 性能要求

| 指标 | 要求 |
|-----|------|
| 地图加载时间 | < 2秒 |
| 定位精度 | 室外 < 5米，室内 < 10米 |
| AI响应时间 | < 3秒 |
| 图片加载 | 缩略图 < 500ms |
| 冷启动时间 | < 3秒 |

### 6.2 兼容性要求

| 项目 | 要求 |
|-----|------|
| Android 版本 | 8.0+ (API 26) |
| 屏幕适配 | 手机、平板 |
| 网络要求 | 4G 及以上 |

### 6.3 安全与隐私

- 位置数据仅本地处理或用户授权后上传
- 好友定位需双方同意
- 用户可随时关闭位置共享
- 敏感操作需二次确认
- 遵循《个人信息保护法》
